# Cybersecurity-Home-Lab-
A controlled lab environment built to simulate real-world attack and defense scenarios using virtual machines.
## Overview
A controlled lab environment built to simulate real-world attack and defense scenarios using virtual machines.

## Tools Used
Kali Linux, Metasploitable, Windows, Metasploit, Wireshark, Nmap

## What I Did
- Performed network reconnaissance using Nmap to identify open ports and services
- Exploited the vsFTPd 2.3.4 vulnerability using Metasploit in a controlled environment
- Captured and analyzed network traffic with Wireshark to identify suspicious activity and attack patterns
- Applied system hardening techniques including firewall configuration, patching, and service reduction

## What I Learned
Gained hands-on experience with the full attack lifecycle — from reconnaissance to exploitation to remediation — in a safe, legal environment.
